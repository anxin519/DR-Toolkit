# DICOM运维工具 - 开发者指南

## 项目概述

这是一个基于Python和Tkinter的DICOM医疗影像运维工具，采用模块化设计，易于维护和扩展。

## 核心模块说明

### 1. 配置管理 (config_manager.py)

**功能：** 管理应用配置，支持自动创建和持久化

**关键类：**
```python
class ConfigManager:
    def __init__(self, config_file="config/app_config.json")
    def load_config()
    def save_config(config=None)
    def get(key, default=None)
    def set(key, value)
```

**使用示例：**
```python
config = ConfigManager()
port = config.get('local_scp.port', 11112)
config.set('local_scp.port', 11113)
```

### 2. 日志系统 (logger.py)

**功能：** 提供分类日志记录，按日期自动分文件

**关键类：**
```python
class Logger:
    @staticmethod
    def get_logger(name, log_dir="logs")
```

**使用示例：**
```python
logger = Logger.get_logger('app')
logger.info("应用启动")
logger.error("发生错误", exc_info=True)
```

### 3. 转发队列 (forward_queue.py)

**功能：** 管理DICOM转发任务，支持失败重试

**关键类：**
```python
class ForwardQueue:
    def add_task(filepath, target_node, source_ae="")
    def mark_success(task_id)
    def mark_failed(task_id, error)
    def get_pending_tasks()
    def retry_task(task_id)
    def start_worker(forward_callback)
```

**使用示例：**
```python
queue = ForwardQueue()
queue.add_task(filepath, target_node)
queue.start_worker(forward_callback)
```

## 开发规范

### 代码风格

1. **命名规范**
   - 类名：大驼峰 (PascalCase)
   - 函数名：小写下划线 (snake_case)
   - 常量：大写下划线 (UPPER_CASE)
   - 私有方法：前缀下划线 (_method)

2. **注释规范**
   ```python
   def function_name(param1, param2):
       """
       函数简短描述
       
       参数:
           param1: 参数1说明
           param2: 数2说明
       
       返回:
           返回值说明
       """
       pass
   ```

3. **导入顺序**
   ```python
   # 标准库
   import os
   import sys
   
   # 第三方库
   import pydicom
   import numpy as np
   
   # 本地模块
   from core.config_manager import ConfigManager
   ```

### 错误处理

1. **使用try-except**
   ```python
   try:
       # 可能出错的代码
       result = risky_operation()
   except SpecificException as e:
       logger.error(f"操作失败: {e}")
       messagebox.showerror("错误", str(e))
   ```

2. **记录日志**
   ```python
   logger.info("开始处理")
   logger.warning("警告信息")
   logger.error("错误信息", exc_info=True)
   ```

### 线程使用

1. **后台任务**
   ```python
   def long_task():
       # 耗时操作
       pass
   
   threading.Thread(target=long_task, daemon=True).start()
   ```

2. **UI更新**
   ```python
   def update_ui():
       # UI更新代码
       pass
   
   root.after(0, update_ui)
   ```

## 添加新功能

### 1. 添加新的DICOM功能

**步骤：**
1. 在 `src/dicom/` 创建新模块
2. 实现功能类
3. 在GUI中调用

**示例：**
```python
# src/dicom/new_feature.py
class NewFeature:
    def __init__(self):
        pass
    
    def do_something(self):
        pass

# 在GUI中使用
from dicom.new_feature import NewFeature

feature = NewFeature()
feature.do_something()
```

### 2. 添加新的标签页

**步骤：**
1. 在主窗口类中添加创建方法
2. 在 `create_ui()` 中注册标签页

**示例：**
```python
def create_new_tab(self):
    """创建新标签页"""
    frame = ttk_boot.Frame(self.notebook)
    
    # 添加控件
    label = ttk_boot.Label(frame, text="新功能")
    label.pack()
    
    return frame

# 在create_ui中
self.notebook.add(self.create_new_tab(), text="🆕 新功能")
```

### 3. 添加配置项

**步骤：**
1. 在 `ConfigManager.DEFAULT_CONFIG` 添加默认值
2. 使用 `config.get()` 读取
3. 使用 `config.set()` 保存

**示例：**
```python
# 在DEFAULT_CONFIG中
"new_feature": {
    "enabled": True,
    "value": 100
}

# 使用
enabled = self.config.get('new_feature.enabled', True)
self.config.set('new_feature.value', 200)
```

## 常见问题

### Q: 如何添加新的窗宽窗位预设？
A: 在配置文件的 `ui_settings.window_presets` 中添加：
```json
"bone": {"center": 400, "width": 1800}
```

### Q: 如何修改日志级别？
A: 在 `logger.py` 中修改：
```python
logger.setLevel(logging.INFO)  # 改为INFO
```

### Q: 如何添加新的匿名化标签？
A: 在 `anonymizer.py` 的 `TAGS_TO_ANONYMIZE` 列表中添加。

### Q: 如何自定义UI主题？
A: 修改 `main_complete.py` 中的：
```python
root = ttk_boot.Window(themename="darkly")  # 更换主题
```
