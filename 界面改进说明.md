# 界面改进说明 v2.1.1

## 更新日期
2025-03-13

## 改进内容

### 1. 📤 发送页面改进

#### 改进1：双击节点填充编辑区
**功能：** 双击节点列表中的节点，自动将节点信息填充到右侧编辑区

**使用方法：**
1. 在左侧节点列表中双击任意节点
2. 节点的名称、AE Title、Host、Port会自动填充到右侧
3. 可以修改信息后点击"更新节点"保存

**使用场景：**
- 快速编辑现有节点
- 复制节点信息创建新节点
- 查看节点详细配置

**操作对比：**

**之前：**
```
1. 记住节点信息
2. 手动输入到编辑区
3. 更新节点
```

**现在：**
```
1. 双击节点
2. 自动填充
3. 修改后更新
```

#### 改进2：单击切换选中状态
**功能：** 单击节点列表第一列（复选框列）切换选中状态

**使用方法：**
- 单击节点行的第一列（☐ 或 ☑）切换选中状态
- 双击节点行填充到编辑区

**交互说明：**
- **单击第一列** → 切换选中状态（☐ ↔ ☑）
- **双击任意位置** → 填充到编辑区

#### 改进3：按钮竖向排列
**功能：** 全选、全不选、测试选中三个按钮改为竖向排列

**布局对比：**

**之前（横向）：**
```
[全选] [全不选] [测试选中]
```

**现在（竖向）：**
```
[全选      ]
[全不选    ]
[测试选中  ]
```

**优势：**
- 节省横向空间
- 按钮更大更易点击
- 布局更整洁

### 2. 📥 接收页面改进

#### 改进：添加自动转发开关
**功能：** 在启动按钮前添加"启用自动转发"勾选框

**界面布局：**
```
┌─────────────────────────────────────────────────┐
│ SCP配置                                         │
│ AE: [DICOM_TOOL] Port: [11112] 存储: [./storage]│
│                                                 │
│ [☐ 启用自动转发] │ [▶ 启动] [⏹ 停止]          │
└─────────────────────────────────────────────────┘
```

**使用方法：**
1. 勾选"启用自动转发"
2. 点击"▶ 启动"启动SCP服务
3. 接收到的文件会自动转发到配置的节点

**功能说明：**
- 勾选后，接收的DICOM文件会自动转发
- 转发规则在"转发"标签页配置
- 转发失败会自动重试（5分钟后）

**应用场景：**
- 接收后自动备份到其他PACS
- 多院区数据自动分发
- 接收后自动归档

## 详细使用说明

### 发送页面完整操作流程

#### 流程1：添加新节点并发送
```
1. 在右侧填写节点信息
   名称: 测试PACS
   AE:   TEST_SCP
   Host: 192.168.1.200
   Port: 104

2. 点击"添加节点"

3. 在左侧列表中单击第一列勾选该节点

4. 添加要发送的文件

5. 点击"发送到选中节点"
```

#### 流程2：编辑现有节点
```
1. 在左侧列表中双击要编辑的节点

2. 右侧自动填充节点信息

3. 修改需要更改的字段

4. 点击"更新节点"

5. 节点信息已更新
```

#### 流程3：批量测试节点
```
1. 点击"全选"按钮（竖向排列的第一个）

2. 点击"测试选中"按钮（竖向排列的第三个）

3. 等待测试完成

4. 查看每个节点的状态（✓ 或 ✗）

5. 取消不可用的节点

6. 发送文件
```

### 接收页面完整操作流程

#### 流程1：普通接收
```
1. 配置AE Title和端口

2. 设置存储路径

3. 点击"▶ 启动"

4. 等待接收文件

5. 查看日志
```

#### 流程2：接收并自动转发
```
1. 先在"转发"标签页配置转发规则

2. 回到"接收"标签页

3. 勾选"启用自动转发"

4. 点击"▶ 启动"

5. 接收的文件会自动转发到配置的节点

6. 查看日志了解转发状态
```

## 技术实现

### 双击填充功能
```python
def load_node_to_edit(self, event):
    """双击加载节点到编辑区"""
    item = self.node_tree.identify_row(event.y)
    if not item:
        return
    
    values = self.node_tree.item(item, 'values')
    
    # 填充到右侧编辑区
    self.node_name.delete(0, 'end')
    self.node_name.insert(0, values[1])  # 名称
    
    self.node_ae.delete(0, 'end')
    self.node_ae.insert(0, values[2])  # AE
    
    # ... 其他字段
```
