# DICOM运维工具 - 快速参考

## 🚀 快速启动

```bash
# 双击运行
启动工具.bat

# 或命令行
python src/main_complete.py
```

## 📤 多节点发送（新功能）

### 快速操作
1. **双击**节点列表中的节点勾选（可多选）
2. 添加文件
3. 点击"发送到选中节点"

### 快捷按钮
- **全选** - 选中所有节点
- **全不选** - 取消所有选择
- **测试选中** - 批量测试连接

### 节点管理
```
添加节点 → 填写信息 → 点击"添加节点"
更新节点 → 选中节点 → 修改信息 → 点击"更新节点"
删除节点 → 选中节点 → 点击"删除节点"
```

## 📥 接收文件

```
配置端口 → 设置存储路径 → 点击"▶ 启动"
```

## ✏️ 编辑文件

```
打开文件 → 查看图像/标签 → 编辑 → 保存
```

### 常用操作
- **匿名化** - 保留病历号后4位
- **修改UID** - 原UID + 时间戳
- **计算年龄** - 自动根据日期计算
- **窗宽窗位** - 滑块调整或点击"肺窗"

## 📁 批量处理

```
选择文件夹 → 等待扫描 → 批量操作 → 导出Excel
```

### 批量操作菜单
- **批量匿名化** - 处理所有文件
- **批量计算年龄** - 仅处理无年龄的
- **批量修改UID** - 保持一致性

## ⚙️ 配置文件

**位置：** `config/app_config.json`

**常用配置：**
```json
{
  "remote_nodes": [...],        // 远程节点列表
  "local_scp": {...},           // 本地SCP配置
  "anonymize": {
    "keep_last_digits": 4,      // 保留位数
    "prefix": "ANON"            // 匿名前缀
  },
  "uid_strategy": {
    "method": "append_timestamp" // UID策略
  }
}
```

## 📝 日志文件

**位置：** `logs/`

**文件：**
- `app_YYYY-MM-DD.log` - 应用日志
- `forward_YYYY-MM-DD.log` - 转发日志
- `scp_YYYY-MM-DD.log` - SCP日志

## 🔍 常见问题

### Q: 如何同时发送到多个PACS？
A: 在节点列表中勾选多个节点，然后点击"发送到选中节点"。

### Q: 如何测试节点连接？
A: 勾选节点后点击"测试选中"按钮。

### Q: 节点配置保存在哪里？
A: `config/app_config.json` 文件中。

### Q: 如何修改匿名化保留位数？
A: 编辑配置文件中的 `anonymize.keep_last_digits`。

### Q: 批量处理会覆盖原文件吗？
A: 是的，会直接修改原文件，建议先备份。

### Q: 如何查看详细日志？
A: 打开 `logs/` 目录下对应日期的日志文件。

## 🎯 使用技巧

### 技巧1：快速测试多个节点
```
1. 点击"全选"
2. 点击"测试选中"
3. 查看哪些节点可用
```

### 技巧2：主备同时发送
```
1. 勾选主PACS和备份PACS
2. 一次发送，两处存储
3. 提高数据安全性
```

### 技巧3：批量处理前预览
```
1. 选择文件夹扫描
2. 查看表格确认文件信息
3. 导出Excel保存记录
4. 再执行批量操作
```

### 技巧4：窗宽窗位快速调整
```
1. 打开图像后自动计算
2. 点击"肺窗"快速切换
3. 或用滑块精细调整
```

### 技巧5：节点分组管理
```
建议命名规范：
- "生产-PACS主"
- "生产-PACS备"
- "测试-PACS"
- "开发-PACS"
```
