# DICOM 运维工具 使用说明

## 启动

```bash
python src/main_complete.py
```
或双击 `启动工具.bat`

---

## 各功能页说明

### 📤 发送
- 右侧填写节点信息后点"添加节点"，支持多节点管理
- 双击节点列表中的行可将参数填入右侧编辑区
- 点击第一列（✓）勾选目标节点，支持全选/全不选
- "测试选中"对勾选节点发送 C-ECHO，失败时显示具体原因
- 添加文件/文件夹后点"发送到选中节点"，支持同时发送到多个节点

### 📥 接收
- 配置本地 AE Title、端口、存储路径后点"▶ 启动"
- 勾选"启用自动转发"后，收到文件会自动加入转发队列，发送到发送页中勾选的节点
- 下方转发队列面板实时显示任务状态，支持手动重试失败任务、清除已完成任务

### 📋 Worklist
- 上半部分：启动本地 Worklist SCP 服务，响应设备的 C-FIND 查询
- 数据管理：手动添加患者信息，或点"生成测试数据"生成10条随机数据
- 下半部分：SCU 查询，向指定服务器发送 C-FIND，支持按患者ID/姓名/模态过滤

### ✏️ 编辑器
- 打开单个 DCM 文件，左侧显示图像，右侧显示全部标签
- 窗宽窗位：首次打开自动调整，可手动拖动滑块或点预设按钮（肺窗/纵隔/骨窗/软组织）
- 仅当文件中无年龄字段时显示"计算年龄"按钮（根据出生日期自动计算）
- 匿名化：保留病历号后4位，其余字段清空
- 修改UID：默认在原UID后追加时间戳，保证唯一性且不超过64字符限制

### 📁 文件浏览
- 选择文件夹后自动扫描所有 .dcm 文件并列表显示
- 点击列标题可排序
- 批量操作：匿名化、计算年龄（仅处理无年龄的文件）、修改UID
- 导出 Excel：将当前列表信息导出为 .xlsx 文件

---

## 中文乱码说明

程序内置编码自动检测，支持 GB18030/GBK/GB2312/UTF-8 等编码的自动识别和修复。
修复后会将 SpecificCharacterSet 更新为 UTF-8，保存后不再出现乱码。

---

## 配置文件

`config/app_config.json` 首次运行自动创建，包含：
- 远程节点列表
- 本地 SCP 配置
- 窗宽窗位预设
- 匿名化规则（保留位数、前缀）
- UID 修改策略
