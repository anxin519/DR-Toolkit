# DICOM运维工具 v2.1.1 - 更新总结

## 更新概述

本次更新主要针对用户界面进行了优化，提升了操作效率和用户体验。

## 核心改进

### ✅ 1. 双击节点填充编辑区

**改进前：**
- 需要手动记住节点信息
- 逐个字段输入到编辑区
- 容易输入错误

**改进后：**
- 双击节点即可自动填充
- 所有字段一键加载
- 快速编辑，减少错误

**使用场景：**
- 修改现有节点配置
- 复制节点创建新配置
- 查看节点详细信息

### ✅ 2. 单击切换选中状态

**改进前：**
- 双击切换选中状态
- 与编辑功能冲突

**改进后：**
- 单击第一列切换选中
- 双击填充到编辑区
- 两个功能互不干扰

**交互逻辑：**
```
单击第一列 → 切换选中状态（☐ ↔ ☑）
双击任意处 → 填充到编辑区
```

### ✅ 3. 按钮竖向排列

**改进前：**
```
[全选] [全不选] [测试选中]  ← 横向排列
```

**改进后：**
```
[全选      ]
[全不选    ]  ← 竖向排列
[测试选中  ]
```

**优势：**
- 节省横向空间
- 按钮更大更易点击
- 视觉更整洁

### ✅ 4. 自动转发开关

**改进前：**
- 自动转发功能不明显
- 需要在配置中设置

**改进后：**
```
[☐ 启用自动转发] │ [▶ 启动] [⏹ 停止]
```

**功能说明：**
- 勾选后自动转发接收的文件
- 转发规则在转发页面配置
- 失败自动重试（5分钟）

## 使用示例

### 示例1：快速编辑节点

```
步骤：
1. 双击"PACS主服务器"节点
2. 右侧自动填充：
   名称: PACS主服务器
   AE:   PACS_SCP
   Host: 192.168.1.100
   Port: 104
3. 修改Host为 192.168.1.200
4. 点击"更新节点"
5. 完成！
```

### 示例2：批量选择和测试

```
步骤：
1. 点击"全选"按钮
2. 点击"测试选中"按钮
3. 查看测试结果
4. 单击不可用节点的第一列取消选择
5. 点击"发送到选中节点"
```

### 示例3：接收并自动转发

```
步骤：
1. 配置转发规则（转发页面）
2. 回到接收页面
3. 勾选"启用自动转发"
4. 点击"▶ 启动"
5. 接收的文件自动转发
```
